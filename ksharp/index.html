K# Documentation


Variables

1. Making a variable

Syntax:

var var_name = value;

Examples:

var a = 1;
var b = 2;
print(a);
print(b);
# This prints 12 on console, see "print and println".

2. Setting a variable

K# is dynamic-typed, so unlike C#, you could set a string to an int.

Syntax: 

a = value; #Assignment
a += value; #Add
a -= value; #Subtract 
a *= value; #Multiply 
...
It also applies to every python operator that has "=" (except :=)

Examples:

var a = 1;
print(a); #1
a += 1;
print(a); #2

3. Type functions

This has all builtin type functions for functions that return None (in-place)

Syntax:

varname append item; #varname.append(item) in py
varname extend iterable; #varname.extend(iterable)
varname remove item; #varname.remove(item)
varname sort; #varname.sort()
varname reverse; #varname.reverse()
varname clear; #varname.clear()
varname insert (index, value); #varname.insert(index, value)
varname add item; #varname.add(item)
varname update item; #varname.update(item)
varname discard item; #varname.discard(item)
varname intersection_update iterable; #varname.intersection_update(iterable)

Examples:

var a = [];
var b = input("Enter a value: "); #input 1
a append 'b';    #for tostring conversion on k#
print(a); #['1'] bcz user input
b = input("Enter a value: "); #input 2
a append b;    #for eval on k#
print(a); #['1', 2]
b = input("Enter a value: "); #input a
a append b;    #error: "a" is not defined

Functions

1. Declaring a function

Syntax:

func funcname() {func_data;}

func funcname(paramname) {func_data;}

func funcname(paramname=defaultval) {func_data;}

func funcname(paramname, anotherparam) {func_data;}

func funcname(paramname, anotherparam=defaultval) {func_data;}

func funcname(paramname, anotherparam=defaultval, anotherparam2=defaultval2) {func_data;}

Example (Area calculator):

func calculatearea(width, height) {
    var a = height * width;
    return a;
}
var w = int(input("Enter width: "));
var h = int(input("Enter height: "));
var area = calculatearea(w, h);
print("Area: ");
println(area);

2. Calling functions

Syntax:

funcname();

funcname(val);

funcname(val1, val2);

funcname(val1, val2, val3); #use many params as you want

Example: See "Area calculator"

3. return

Syntax:

return value;

Example: See "Area calculator"

Control flows

1. if / else

Syntax:

if (condition) {codegoeshere;}

if (condition) {codegoeshere;} else {anothercodegoeshere;}

Examples:

var a = 1;
if (a == 12) {
    println("Hi12");
}
if (a > 12) {
    print("Hi");
}
else {
    print("Hi2");
}

2. while

Syntax:

while (condition) {codegoeshere;}

Example:

var a = 2;
while (a == 2) {
    print(a);
} # forever print 2

3. for

Syntax:

for (newvarname, iterable) {codegoeshere;}

Example:

var a = [1, 2, 3, 4, 5];
for (b, a) {
    print(b);
} # 12345

4. until

Syntax:

until (condition) {codegoeshere;}

Example:

var a = 2;
until (a != 2) {
print(a);
a += 1;
} # prints 2 once

Modules

1. Creating them

To create a module, first create a file with extension .kshp, then write the module.

2. Using them

Syntax:

using "path/to/module";

Then use the module by functions (without "modulename.functionname") or vars.

Basically, filedata = module + mainprog

print and println

So basically, print(text) is print(text, end='') in python

println(text) however is print(text) in python
